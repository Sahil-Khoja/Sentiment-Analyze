# Brand Review Sentiment Analyzer 📊

A frontend-based NLP application that analyzes customer reviews and classifies them as Positive, Negative, or Neutral. Built entirely with Vanilla JavaScript, the application performs text preprocessing, tokenization, stemming, sentiment scoring, and word frequency analysis directly in the browser without requiring a backend.

## 🚀 Features

- Brand-wise sentiment analysis
- Positive, Negative, and Neutral classification
- Custom NLP processing pipeline
- Stop-word removal and tokenization
- Stemming and negation handling
- Interactive analytics dashboard
- Review filtering by sentiment
- Word frequency visualization
- Upload review datasets via JSON, CSV, or TXT files
- Pure frontend implementation (no backend required)

## 🛠️ Tech Stack

- HTML5
- CSS3
- Tailwind CSS
- JavaScript (ES6 Modules)
- Vitest
- Fast-Check
